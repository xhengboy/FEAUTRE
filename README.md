Image Enhancement and Image Fusion
[Enhancement ]

1. 影像增強技術，使用 OpenCV、PIL 進行影像處理。

2.具備影像品質優化能力，包含：亮度、降噪、對比度調整、紋理細節提升

3.設計基於深度學習的影像增強模型（如 CNN-based enhancement），提升下游任務（如分類、瑕疵檢測）的表現。

[Fusion]

1.具備多來源影像融合概念，能整合不同影像資訊提升特徵表現能力。

2.整合來自不同影像來源的特徵資訊，相較於單一影像來源，多元影像數據能提供更為全面且關鍵的資訊，進而大幅強化檢測系統的穩定性與準確性。

Machine Learning
1.使用機器學習與深度學習模型開發流程，包含資料前處理、模型訓練、驗證與測試。

2.使用 Python 機器學習相關套件（NumPy、Matplotlib、PyTorch），能完整建構訓練與評估流程。

3.具備模型訓練與驗證經驗，包含Dataset建立、K-Fold Cross Validation、評估指標（Accuracy、F1 Score、Precision、Recall、Confusion Matrix）。

4.獨立完成從資料處理 → 模型訓練 → 評估 → 結果分析的完整。
